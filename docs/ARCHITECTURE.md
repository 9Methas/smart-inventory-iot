# Architecture Documentation

## System Overview

Smart Inventory & Environment Monitor เป็นระบบ Mono-repo ที่แยก Frontend และ Backend ออกจากกันอย่างชัดเจน

## Tech Stack

- **Frontend**: Next.js 14+ (App Router), TypeScript, Tailwind CSS
- **Backend**: Node.js, Express, TypeScript
- **Shared**: TypeScript types และ utilities
- **IoT**: MQTT, WebSocket support
